Research


Patatap
https://patatap.com/
Patatap merges sound and visuals in a playful, immediate way. Each keystroke triggers an audiovisual reaction, showing how interaction can feel create an immersive experience. It showcases minimal design with a highly engaging outcome.

Fabio Formato
https://fabioformato.com/portfolio/
This portfolio exemplifies clean, responsive interaction paired with smooth transitions. This work showcases how animation and interaction can elevate user experience without overwhelming the content. It encourages thoughtful pacing and intuitive navigation which is necessary for interactive sites.

Typatone
https://typatone.com/
Typatone turns typing into ambient music, showing how text input can be repurposed for entertainment! It expands the function of a common interface and invites the user into a meditative, sound-based interactive design.

Radar
https://lab.hakim.se/radar/
Radar is a minimalist music sequencer using spatial placement and loops. It highlights how simple geometric shapes and sound can create interesting points in web design when manipulated properly.
